# Стиль кода — FG SEVER

## TypeScript
- Строгий режим (`strict: true`), никаких `any`
- Явные типы для пропсов компонентов (интерфейсы, не `type`)
- Типы экспортировать из `src/data.ts`

## Именование
- Компоненты: PascalCase (`ProductCard`, `OrderModal`)
- Хуки: camelCase с префиксом `use` (`useProducts`, `useGroups`)
- Файлы компонентов: PascalCase.tsx
- Файлы хуков: camelCase.ts

## Комментарии
- Писать **только на русском**
- Только если логика неочевидна
- Не комментировать очевидный код

## Импорты
- Сначала внешние библиотеки, затем внутренние
- Абсолютные пути от `src/` (через алиас `@/` если настроен)

## Форматирование
- Отступы: 2 пробела
- Точки с запятой: да
- Одинарные кавычки для строк
